The workings

Cross-device sync

Sync uses a private GitHub Gist. Create a Personal Access Token with the gist scope at github.com/settings/tokens, paste it below, and press Sync now. A new gist is created on first sync; afterwards, the same token on another device pulls your data down.

On a new device: paste an existing Gist ID, then press Pull from cloud.
